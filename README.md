# zheye

仿知乎专栏项目，可以实现登录注册、上传文章和图片、查看专栏等功能

## 在线访问

[zheye (cloudendpoint.cn)](https://zhihu.web.cloudendpoint.cn/)

## 技术栈

+ Vue
+ TypeScript
+ Vuex
+ Vue-Router
+ BootStrap

## 开发说明

开发过程的学习笔记在 ./docs 目录下，与代码同步提交

提交规范：

+ feat：新功能
+ update：更新某功能
+ fix：修补某功能的 bug
+ refactor：重构某个功能
+ optimize: 优化构建工具或运行时性能
+ style：仅样式改动
+ docs：仅文档新增/改动
+ chore：构建过程或辅助工具的变动
